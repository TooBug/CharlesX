# CharlesX

HTTP/HTTPS代理工具，类似Charles、Fiddler。

## 特性

- 支持HTTP/HTTPS代理
- 支持自定义hosts

## 安装

```sh
npm install -g charlesx
```

## 使用

```sh
# 以默认端口8888启动
charlesx

# 以端口9999启动
charlesx -p 9999

# 自定义hosts
charlesx -s www.toobug.net=127.0.0.1 -s toobug.net=127.0.0.1
```
